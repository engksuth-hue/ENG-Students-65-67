# แดชบอร์ดจำนวนนักศึกษา (คณะวิศวกรรมศาสตร์) — เวอร์ชันละเอียด (รายสาขาครบ)

จุดเด่น
- แท็บ **ภาพรวม**: กราฟแท่ง (ปี 65–67) + กราฟวงกลม (ปีที่เลือก)
- แท็บ **รายสาขา (ครบ)**: ตารางแสดง “ทุกสาขา” + ค้นหา + จัดเรียง + รวมท้ายตาราง + กราฟแท่งรายสาขา

## ติดตั้งและรัน
```bash
npm install
npm run dev
```

## ข้อมูล JSON (จาก Power BI)
แก้ได้ที่ `public/data/students_65_67.json`

### สำคัญ: ให้สาขาแสดงครบ
ในแต่ละปี ต้องมี `by_department` และใส่รายการสาขาทั้งหมด (หนึ่งสาขา = 1 แถว) พร้อมคอลัมน์ครบ:
- intake_plan, new_intake, retained, not_open, total

## .env (ถ้าโหลดจาก URL ภายนอก)
```bash
cp .env.example .env
```
แล้วแก้ `VITE_DATA_URL=...`
